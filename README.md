## Процедура запуска автотестов:
1. Скачать проект по заданию;
2. Развернуть проект в запущенной программе Android Studio;
3. Подключить тестовое мобильное устройство или эмулятор устройства;
4. При завершении подключения зависимостей и написания автотестов выполнить сборку проекта командой `./gradlew build` в терминале проекта из корневой папки;
5. Тестовые классы находятся по пути `fmh_android_15_03_24\app\src\androidTest\java\ru\iteco\fmhandroid\ui`;
6. Выполнить команду `./gradlew test`, запустив выполнение автотестов. Запуск тестов также можно осуществить из тестового класса одиночно зеленой кнопкой Run;
7. В терминале перейти в корневую директорию проекта и выполнить команду для сборки и запуска всех авто-тестов:
8. Для получения и сохранения отчетов тестов перейти в интерфейсе Android Studio в Device Explorer по пути `\data\data\ru.iteco.fmhandroid\files\allure-results`;
9. Сохранить отчеты об автотестах папкой(архивом) в корне проекта;
10. При установленном Allure и наличии папки allure-results в корне проекта командой `allure serve` отчет генерируется в браузере.
